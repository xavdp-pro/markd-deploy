# Changelog

All notable changes to MarkD will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-20

### Added
- Initial release of MarkD
- Document management with hierarchical folder structure
- Real-time Markdown editor with preview
- Password vault with AES-256 encryption
- Workspace-based organization
- Role-based access control (Admin, Write, Read)
- User and group management
- Document locking system for collaborative editing
- Drag and drop support for documents and folders
- Full-text search for documents and passwords
- Dark mode support
- Responsive design for mobile and desktop
- JWT-based authentication
- WebSocket support for real-time updates
- RESTful API with FastAPI
- React frontend with TypeScript
- PM2 configuration for production deployment

### Security
- AES-256 encryption for password storage
- JWT token-based authentication
- Secure password hashing with bcrypt
- SQL injection protection
- XSS protection
- CSRF protection
